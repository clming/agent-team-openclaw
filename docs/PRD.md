# 微信登录功能 PRD

## 1. 需求概述

### 背景
随着移动互联网的发展，微信已成为中国用户最常用的社交和通讯工具。为降低用户注册门槛、提升转化率，需要在应用中集成微信登录功能，让用户能够一键授权登录，无需记忆额外账号密码。

### 目标用户
- **C 端用户**：使用移动 APP 和 WEB 端的普通用户，希望快速注册/登录
- **运营人员**：需要获取用户基本信息进行用户画像分析
- **开发人员**：需要统一的身份认证接口

### 产品目标
1. 降低注册门槛，提升新用户转化率 30%+
2. 简化登录流程，减少登录耗时至 5 秒内
3. 获取用户授权信息，完善用户画像

## 3. 功能列表

| 优先级 | 功能 | 描述 |
|:------:|------|------|
| **Must** | 微信授权登录 | 调用微信 OAuth 2.0 获取用户授权码，换取 access_token 和用户信息 |
| **Must** | 账号绑定/解绑 | 支持微信账号与系统账号的绑定和解绑操作 |
| **Must** | 登录状态管理 | 维护用户登录态，支持 token 刷新和过期处理 |
| **Must** | 错误处理 | 处理微信授权失败、网络异常、账号已绑定等异常情况 |
| **Should** | 多端支持 | 同时支持移动 APP (SDK) 和 WEB 端 (扫码/H5) 微信登录 |
| **Should** | 用户信息同步 | 同步微信昵称、头像、性别、地区等基本信息 |
| **Should** | 登录日志 | 记录微信登录行为，用于安全审计和数据分析 |
| **Could** | 微信手机号授权 | 获取用户绑定的手机号 (需额外授权) |
| **Could** | 一键登录优先 | 未安装微信时自动降级到其他登录方式 |
| **Won't** | 微信分享功能 | 本次不包含分享到微信的功能 |
| **Won't** | 微信支付集成 | 本次不包含支付相关功能 |

## 4. 验收标准

1. **Given** 用户未注册且未安装微信  
   **When** 用户点击微信登录按钮  
   **Then** 提示"未检测到微信应用"并引导下载或显示其他登录方式

2. **Given** 用户已安装微信且未授权过  
   **When** 用户点击微信登录按钮  
   **Then** 跳转微信授权页面，显示授权范围 (昵称、头像等)

3. **Given** 用户已完成微信授权  
   **When** 授权成功返回应用  
   **Then** 自动完成登录并跳转至首页，全程不超过 5 秒

4. **Given** 用户微信已绑定其他账号  
   **When** 用户尝试绑定新账号  
   **Then** 提示"该微信已绑定其他账号"并提供解绑或客服入口

5. **Given** 用户已登录且想解绑微信  
   **When** 用户在设置中点击解绑微信  
   **Then** 需验证当前登录密码后解绑，且账号必须有其他登录方式

6. **Given** 用户 access_token 过期  
   **When** 应用检测到 token 失效  
   **Then** 自动刷新 token 或引导重新授权，用户无感知

7. **Given** WEB 端用户扫码登录  
   **When** 用户使用微信扫描二维码  
   **Then** 手机确认登录后，WEB 端自动完成登录并跳转

## 5. 技术依赖

### 平台依赖
- **微信公众号/开放平台**
  - 需注册微信开放平台账号 (open.weixin.qq.com)
  - 创建移动应用 (APP) 和网站应用 (WEB)
  - 获取 AppID 和 AppSecret
  - 完成开发者资质认证

### 协议依赖
- **OAuth 2.0 授权**
  - Authorization Code 授权模式
  - access_token 获取与刷新
  - 用户信息接口 (snsapi_userinfo)
  - 静默授权 (snsapi_base) 用于获取 openid

### 后端接口
| 接口 | 方法 | 描述 |
|------|------|------|
| `/api/auth/wechat` | POST | 发起微信登录，返回授权 URL 或 SDK 参数 |
| `/api/auth/wechat/callback` | GET/POST | 微信回调处理，换取 token 并登录 |
| `/api/auth/wechat/bind` | POST | 绑定微信账号到现有用户 |
| `/api/auth/wechat/unbind` | POST | 解绑微信账号 |
| `/api/auth/wechat/refresh` | POST | 刷新 access_token |

### 前端依赖
- **移动 APP**
  - 微信开放平台 SDK (iOS/Android)
  - 配置 URL Scheme 或 Universal Links
- **WEB 端**
  - 微信扫码登录 JS SDK
  - 或 H5 跳转授权模式

### 安全要求
- AppSecret 仅存储于后端，严禁前端暴露
- access_token 加密存储，设置合理过期时间
- 回调 URL 白名单校验
- 防重放攻击 (state 参数校验)
